Loan Approval Prediction Using Machine Learning

Project Overview

This project aims to predict loan approval using machine learning techniques. By analyzing historical data and training a model, we can make predictions on whether a loan application will be approved or not.

Installation

git clone https://github.com/ShravyaMedipally/Loan-Prediction-using-ML.git
cd loan-approval-prediction
pip install -r requirements.txt

Usage

python predict.py --input data/input.csv --model models/loan_approval_model.pkl
